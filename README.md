# CS50P Introduction to Python

This repository contains my practice files, exercises, and problem set solutions for Harvard's CS50P: Introduction to Programming with Python.

## Repository Structure

- `Conditionals/` — exercises for conditional statements
- `Funtions, Variables/` — practice files for variables and functions
- `loops/` — exercises and programs for loops
- `pset 0/` — problem set 0 solutions
- `pset1/` — problem set 1 solutions
- `pset2/` — problem set 2 solutions

## Topics Covered

- Variables
- Functions
- Conditionals
- Loops
- Input and output
- String manipulation
- Basic problem solving in Python

## How to Run

To run any Python file, use:

```bash
python path/to/file.py
```

For example:

```bash
python loops/bee.py
```
